# openYuanrong Data System 特性树（Agent 读本）

本文档由源表 `[workspace/reliability/feature_tree.txt](../../workspace/reliability/feature_tree.txt)` 转换而来，供 Agent 理解 **yuanrong-datasystem / openYuanrong 数据系统** 能力边界。已剔除 **上线时间** 列。

---

## 表头说明


| 列名            | 含义与用法                                                     |
| ------------- | --------------------------------------------------------- |
| **维度**        | 能力大类：基本功能、性能、扩展性、构建、运维、可用性、可靠性等。同一维度下多行若左侧为空，表示与上一行同属该维度。 |
| **特性**        | 维度下的功能分组。左侧为空时表示延续上一行的特性分组。                               |
| **子特性**       | 可验收的具体能力点或行为描述。                                           |
| **可靠性测试是否覆盖** | `Y` 表示源表标注为已覆盖；空表示未标注。**不**等价于「无测试」，分析缺口时需对照代码与测试。        |
