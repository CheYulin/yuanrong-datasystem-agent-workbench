# URMA/TCP 定界修复计划

本目录集中存放“修复 URMA 和 TCP 相关定界问题”的计划类文档，避免与调用链、证据类文档混放。

## 目录内容

- `kv-client-可观测与可靠性-问题分析与代码修改计划.md`  
  说明问题清单、修改方案、分阶段执行计划。
- `kv-client-URMA环境验证执行清单.md`  
  说明有 URMA 环境时的验证项、执行步骤和结果回填模板。
- `issue-rfc-kvclient-urma-tcp-可观测与可靠性整改.md`  
  GitCode Issue / RFC 文案模板（可直接粘贴创建 issue）。
- `pr-description-kvclient-urma-tcp-可观测与可靠性整改.md`  
  GitCode PR 描述模板（可直接粘贴创建 PR）。

## 适用场景

- 需要评审修复需求与实施路径时；
- 需要安排具备 URMA 环境同事进行验证时；
- 需要回溯 TCP/URMA 定界整改范围时。
