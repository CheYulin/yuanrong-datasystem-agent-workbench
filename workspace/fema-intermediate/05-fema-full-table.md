# FMEA 故障分析表（基于 fault.template）

## 使用说明

- **故障域分类**: OS层 / URMA层 / 组件层 / 用户层 / 内部错误
- **定位定界**: 通过错误码、日志关键字、调用链进行区分
- **代码证据**: 详见 `code-evidence/` 目录

---

## FMEA 表格 (CSV格式)

```
一级对象,二级对象,三级对象,故障模式,可能原因,故障影响,是否中断业务,严酷度,故障模式备注,故障预防方法(预期),故障预防方法(版本现状),故障检测方法(预期),故障检测方法(版本现状),故障处理方法（预期）,故障处理方法(版本现状),故障定位方法（预期）,故障定位方法（版本现状）,改进建议,故障管理措施备注
```
